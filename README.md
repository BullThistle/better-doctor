Doctor Lookup
======================

#### By Rafael Furry

This is a web app which utilizes the Better Doctor API to find doctors in the greater Seattle area.

Installation
------------

Clone repository
```
$ git clone https://github.com/BullThistle/better-doctor.git
```

Install required dependencies
```
$ npm install
```
```
$ bower install
```

Run local server
```
$ gulp serve
```
